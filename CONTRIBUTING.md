# Contributing
OCN Standard: Conventional Commits, black+ruff+mypy, 80% coverage, squash-merge, MASTER VIEW approval.

## Commit Convention
- feat: new feature  |  fix: bug fix  |  docs: documentation  |  test: tests  |  chore: maintenance

## Process
1. Fork → feature branch → PR against main
2. All checks must pass (lint, test, security)
3. MASTER VIEW review required for merge
